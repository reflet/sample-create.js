# create.js - サンプル

## ローカル環境
dockerでnginxを起動してwebサーバを構築します。
```
$ docker-compose build
$ docker-compose up -d
```

## 動作確認
ブラウザで閲覧してみる。
```
$ open http://localhost:8080
```
